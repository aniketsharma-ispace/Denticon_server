Privacy Policy

Last updated: May 26, 2025

This Privacy Policy describes how Insurance Auditor Pro ("the Extension") handles information when installed in the Google Chrome browser. By using this Extension, you agree to the practices described in this policy.

Summary: This extension does not collect, transmit, or share any data with external servers. All data processed by this extension stays entirely on your local device.

1. What Data Does the Extension Access?

Insurance Auditor Pro reads the following information from supported dental insurance portals when explicitly triggered by the user:

  • Personally Identifiable Information (PII): Patient name, date of birth, email address, and phone number as displayed on the portal.
  • Health Information: Dental insurance plan details, coverage percentages, procedure categories, and benefits information.
  • Financial Information: Account balances, estimated insurance/patient portions, deductibles, and annual maximums as displayed on the portal.
  • Website Content: Text content from dental portal pages (Denticon, MetLife, Cigna) required to extract the above data.

2. How Is Data Used?

All data accessed by the Extension is used solely for the following purposes:

  • Temporarily storing captured insurance data in the browser's local storage (chrome.storage.local) while the user navigates between portals.
  • Compiling and exporting the data as a structured JSON file to the user's local device for audit and reconciliation purposes.

3. Data Storage & Retention

  • Data is stored only in the user's local browser storage (chrome.storage.local).
  • Data is automatically purged from local storage immediately after the user downloads the JSON export.
  • No data is retained beyond a single audit session.

4. Data Sharing & Transmission

This extension does NOT:
  • Send any data to external servers, APIs, or third parties.
  • Sell or transfer user data to any third party.
  • Use data for any purpose unrelated to the extension's core insurance auditing function.
  • Use data to determine creditworthiness or for lending purposes.

5. Permissions Used

  • storage: Used to temporarily hold audit data in local browser storage between portal sessions.
  • activeTab: Used to detect which portal the user is currently viewing, to display correct status and route commands.
  • downloads: Used to save the compiled JSON audit file to the user's local machine.
  • Host permissions (denticon.com, planetdds.com, metlife.com, cigna.com): Required for content scripts to read insurance data from the supported portals.

6. Supported Portals

The Extension only operates on the following dental insurance portals:

  • Denticon / PlanetDDS (denticon.com, planetdds.com)
  • MetLife Dental Provider Portal (dentalprovider.metlife.com)
  • Cigna for HCP (cignaforhcp.cigna.com)

7. Children's Privacy

This Extension is intended for use by dental professionals only and is not directed at individuals under the age of 13. We do not knowingly collect data from children.

8. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with an updated "Last updated" date. Continued use of the Extension after changes constitutes acceptance of the updated policy.

9. Contact

If you have any questions about this Privacy Policy or the Extension's data practices, please contact the developer through the Chrome Web Store listing page.
